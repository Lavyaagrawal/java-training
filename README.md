# Smart Traffic Monitoring System

## 📌 Description

This project detects vehicle speed violations and stores them in a database.

## ⚙️ Technologies Used

* Java (Core + Streams)
* JDBC
* MySQL
* Swing (Desktop UI)
* Spring Boot + Thymeleaf (Web UI)

## 🚀 Features

* Detect speed violations
* Store data in database
* View data in dashboard
* Search by vehicle ID
* Filter by zone

## ▶️ How to Run

### Desktop App

Run `SmartTrafficSystem.java`

### Web App

Run `SmartTrafficWebApplication.java`
Open: http://localhost:8080

## 🗄️ Database

Create database `traffic_db` and table `violations`
